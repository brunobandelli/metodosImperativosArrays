Um array é uma estrutura de dados que permite armazenar uma coleção de elementos sob um único nome. Esses elementos podem ser de qualquer tipo de dado, como números, strings, objetos, ou até mesmo outros arrays. Os arrays em JavaScript são indexados numericamente, o que significa que cada elemento é acessado através de um número chamado índice.

Os arrays em JavaScript são flexíveis e dinâmicos, o que significa que você pode adicionar ou remover elementos a qualquer momento


//Exemplo simples

// Criando um array de números
let numeros = [10, 20, 30, 40];

// Acessando elementos do array usando índices
console.log(numeros[0]);  // Saída: 10
console.log(numeros[2]);  // Saída: 30

// Modificando um elemento do array
numeros[1] = 50;
console.log(numeros);  // Saída: [10, 50, 30, 40]
